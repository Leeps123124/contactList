# contactList — Практическая №2 (Vanilla JS)

Цель проекта — попрактиковаться с рендером списка на ванильном JavaScript (DOM + события).

## Реализовано
- Рендер контактов из массива на страницу
- Добавление контакта через инпуты (рендер по вводу)
- Удаление одного контакта
- Удалить все контакты
- Простая валидация: если поля пустые/введено неправильно — показывается ошибка

## Пока не реализовано (план)
- Поиск по контактам
- Сортировка/группировка по алфавиту

> Проект не финальный: делался в первую очередь ради практики рендера на vanilla JS
